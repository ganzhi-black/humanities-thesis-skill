# 论文格式规范

本文件覆盖中文人文社科论文的排版格式、脚注引用、参考文献等规范。不同学校和期刊的具体要求可能有差异，以目标院校/期刊的最新版式指南为准。以下是通行的基准规范。

---

## 一、页面设置与字体字号

### 页面设置

| 项目 | 规范 |
|------|------|
| 纸张 | A4（210mm × 297mm） |
| 页边距 | 上 2.5cm、下 2.5cm、左 3.0cm、右 2.5cm |
| 行距 | 正文 1.5 倍行距；注释和参考文献单倍行距 |
| 页码 | 页脚居中，摘要和目录用罗马数字，正文从第 1 页起用阿拉伯数字 |
| 装订 | 左侧装订，预留 0.5cm 装订线 |

### 字体字号

| 部位 | 中文 | 英文/数字 | 字号 |
|------|------|----------|------|
| 论文标题 | 黑体 | Times New Roman 加粗 | 小二号（18pt） |
| 章标题（第一章、第二章） | 黑体 | Times New Roman 加粗 | 三号（16pt） |
| 节标题（1.1、1.2） | 黑体 | Times New Roman 加粗 | 四号（14pt） |
| 小节标题（1.1.1） | 黑体 | Times New Roman 加粗 | 小四号（12pt） |
| 正文 | 宋体 | Times New Roman | 小四号（12pt） |
| 摘要正文 | 宋体 | Times New Roman | 小四号（12pt） |
| "摘要"二字 | 黑体 | — | 三号（16pt） |
| 脚注 | 宋体 | Times New Roman | 小五号（9pt） |
| 参考文献条目 | 宋体 | Times New Roman | 五号（10.5pt） |
| 页眉 | 宋体 | Times New Roman | 五号（10.5pt） |
| 页码 | — | Times New Roman | 小五号（9pt） |

### 特殊排版

- **引用块**（超过 3 行的直接引文）：左缩进 2 字符，字号不变或改为五号，不加引号，上下各空一行
- **图表标题**：图标题在图下方，表标题在表上方，五号黑体居中
- **目录**：标题层级缩进，页码右对齐，用点线连接

## 二、标题编号体系

人文社科论文常用两种编号体系：

### 体系 A：传统中文体系（学位论文常用）

```
第一章  xxxxxx
  第一节  xxxxxx
    一、xxxxxx
      （一）xxxxxx
        1. xxxxxx
```

### 体系 B：数字编号体系（期刊论文常用）

```
1  xxxxxx
  1.1  xxxxxx
    1.1.1  xxxxxx
```

选择哪种体系取决于目标院校/期刊的要求。同一篇论文中只使用一种体系，不要混用。

## 三、脚注/尾注引用格式

人文社科论文通常使用**脚注**（页下注），而非文内夹注。以下是 GB/T 7714 和 Chicago 两种常见格式。

### GB/T 7714 脚注格式（中文期刊和学位论文最常用）

**专著**：
```
① 作者.书名[M].出版地:出版社,出版年:页码.
例：① 福柯.规训与惩罚[M].刘北成,杨远婴,译.北京:三联书店,2003:225.
```

**期刊论文**：
```
② 作者.文章标题[J].期刊名,年,卷(期):页码.
例：② 陈思和.中国当代文学中的创伤叙事[J].文学评论,2018(3):5-15.
```

**学位论文**：
```
③ 作者.论文标题[D].城市:学校名,年份:页码.
例：③ 张三.鲁迅小说的叙事伦理研究[D].北京:北京大学,2020:45.
```

**论文集中的文章**：
```
④ 作者.文章标题[C]//编者.论文集名.出版地:出版社,年:页码.
例：④ 王德威.被压抑的现代性[C]//李欧梵,编.现代性的追求.台北:麦田出版,2000:31-56.
```

**外文文献**（在中文论文中引用时）：
```
⑤ 作者.书名[M].出版地:出版社,年:页码.
例：⑤ Benjamin, Walter. Illuminations[M]. New York: Schocken Books, 1969: 257.
```

**再次引用同一文献**：
```
⑥ 作者.书名[M].页码.
例：⑥ 福柯.规训与惩罚[M].230.
（省略出版信息，只保留作者、书名和新的页码）
```

### Chicago 脚注格式（英文投稿和海外汉学期刊常用）

**专著首次引用**：
```
1. Walter Benjamin, Illuminations, trans. Harry Zohn (New York: Schocken Books, 1969), 257.
```

**期刊论文首次引用**：
```
2. Rey Chow, "Rereading Mandarin Ducks and Butterflies," Cultural Critique 45 (2000): 69.
```

**再次引用（同一文献）**：
```
3. Benjamin, Illuminations, 261.
```

**紧接上条引用同一文献**：
```
4. Ibid., 263.
```

## 四、参考文献格式

参考文献列在论文末尾，与脚注的格式略有不同。

### GB/T 7714 参考文献格式

按文献类型标注 [M]专著 [J]期刊 [D]学位论文 [C]论文集 [N]报纸 [EB/OL]网络文献。

**排列顺序**：中文文献在前，外文文献在后。中文按作者姓氏拼音排列，外文按作者姓氏字母排列。

```
[1] 本雅明.启迪:本雅明文选[M].张旭东,王斑,译.北京:三联书店,2008.
[2] 陈思和.中国当代文学中的创伤叙事[J].文学评论,2018(3):5-15.
[3] 福柯.规训与惩罚[M].刘北成,杨远婴,译.北京:三联书店,2003.
[4] Benjamin, Walter. Illuminations[M]. New York: Schocken Books, 1969.
[5] Chow, Rey. Rereading Mandarin Ducks and Butterflies[J]. Cultural Critique, 2000(45): 69-93.
```

### Chicago 参考文献格式

注意与脚注的差异：姓在前，名在后；句号代替逗号；不标页码（除期刊论文标页码范围）。

```
Benjamin, Walter. Illuminations. Translated by Harry Zohn. New York: Schocken Books, 1969.
Chow, Rey. "Rereading Mandarin Ducks and Butterflies." Cultural Critique 45 (2000): 69–93.
```

### MLA 格式（文学研究英文投稿常用）

```
Benjamin, Walter. Illuminations. Translated by Harry Zohn, Schocken Books, 1969.
Chow, Rey. "Rereading Mandarin Ducks and Butterflies." Cultural Critique, no. 45, 2000, pp. 69–93.
```

## 五、引用格式速查

根据目标期刊/学校的要求选择格式：

| 场景 | 推荐格式 | 注释方式 |
|------|---------|---------|
| 中文学位论文 | GB/T 7714 | 脚注 |
| 中文核心期刊 | GB/T 7714（部分期刊有变体） | 脚注或文末 |
| CSSCI 期刊 | GB/T 7714 | 脚注为主 |
| 英文人文学科期刊 | Chicago（Notes-Bibliography） | 脚注 |
| 英文文学期刊 | MLA | 文内夹注 |
| 英文社科期刊 | APA 或 Chicago（Author-Date） | 文内夹注 |
| 海外汉学期刊 | 视期刊而定（多为 Chicago） | 脚注 |

## 六、常见格式错误

| 错误 | 正确做法 |
|------|---------|
| 中文标点和英文标点混用 | 中文语境用中文标点（全角），英文引文内用英文标点（半角） |
| 书名号内嵌套书名号 | 外层用《》，内层用〈〉 |
| 脚注编号后有空格 | ① 紧跟内容，不加空格；或根据学校模板决定 |
| 参考文献缺少文献类型标识 | [M] [J] [D] 等不可省略（GB/T 7714） |
| 译著只写原作者不写译者 | 必须标注译者姓名 |
| 引用网页不标注访问日期 | [EB/OL] 类文献必须标注访问日期 |
| 同一文献重复出现在参考文献列表中 | 无论引用多少次，参考文献列表中只出现一次 |
| 正文引用的文献不在参考文献列表中 | 正文脚注中出现的所有文献都应收入参考文献 |
