# 学术英文翻译与投稿准备

---

## 摘要英译

中文摘要翻译为英文时，遵循以下原则：

1. **不是逐句翻译，而是重写**：英文学术摘要的句式习惯和中文不同。中文摘要常用"本文认为……""本文论证……"，英文更倾向"This article argues that…""By examining…, this study reveals…"
2. **术语一致性**：同一个概念在全文中只用一个英文对应词，不要换来换去。查阅 `terminology-bilingual.md`（学术术语对照表）获取常见理论术语的标准英译
3. **关键词英译**：关键词直接使用学界通行译法，不要自己造词。如果某个概念没有通行英译，保留原文拼音并在括号内解释

### 英文摘要常用句型

| 功能 | 句型 |
|------|------|
| 提出问题 | This article examines / investigates / explores… |
| 交代方法 | Drawing on [theorist]'s concept of [concept], this study… |
| 陈述论点 | This article argues that… / I contend that… |
| 介绍章节 | The first section analyzes… / Chapter 2 turns to… |
| 总结结论 | By doing so, this study reveals… / This analysis ultimately suggests… |
| 指出意义 | These findings contribute to… / This study sheds light on… |

## 标题英译

- 学术论文英文标题通常采用"主标题：副标题"格式
- 主标题点明核心论点或核心概念，副标题交代研究对象和方法
- 避免过长，控制在 15 个词以内

**示例**：
- 中文：论鲁迅小说中暴力书写的寓言结构
- 英文：Allegories of Violence: Narrative Structure in Lu Xun's Fiction

---

## 理论术语翻译原则

- **已有通行英译的**：直接使用（如"延异"→ différance，"规训"→ discipline）
- **中国特有概念的**：优先查找海外汉学界的通行译法，没有的话用拼音+解释（如"气韵" → qiyun (spirit resonance)）
- **需要回译的**：西方理论经中文学界转化后产生的特殊用法，翻译时要回到原文术语，同时标注中文学界的特殊内涵

翻译时如果遇到不确定的术语，利用联网搜索能力查找该术语在英文学术文献中的标准用法；如果当前环境不支持联网，建议用户在 Google Scholar 中搜索确认。

详细术语对照请查阅 `terminology-bilingual.md`（学术术语对照表）。

## 投稿准备检查清单

| 检查项 | 要求 |
|--------|------|
| 英文摘要字数 | 符合目标期刊要求（通常 150-300 词） |
| 关键词数量 | 通常 4-6 个 |
| 作者信息 | 英文姓名拼写、机构名称规范 |
| 参考文献格式 | 符合目标期刊要求（见 `formatting-guide.md`） |
| 图表标注 | 英文标题、编号与正文对应 |
| 致谢 | 基金项目编号、指导老师等 |
| Cover Letter | 简要说明论文主题、创新点、适合该刊的理由 |
