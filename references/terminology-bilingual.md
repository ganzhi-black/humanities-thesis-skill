# 人文社科学术术语对照表

本文件收录人文社科论文写作和翻译中常用的理论术语。按理论家/学派分组，每个条目包含：中文译名、原文（法/德/英）、英文通行用法、使用语境提示。

翻译原则：优先使用学界已有的通行译法。如果某个术语在不同学者的翻译中有差异，标注主要的几种译法并说明各自的学术脉络。

---

## ⚠ 查不到的术语怎么办

本表无法穷尽所有术语。遇到表中没有的术语时，按以下顺序处理：

1. **联网搜索**：如果当前环境支持联网，搜索 `"术语中文" site:scholar.google.com` 或 `"术语中文" English translation academic`，从英文学术论文中找通行译法
2. **查询 OpenAlex / Semantic Scholar**：用中文术语搜索相关论文，从英文论文标题和摘要中提取对应英文表述
3. **回溯原文**：如果是西方理论经中文转化的概念，先查找原始理论家的母语用词，再看英文学界如何翻译
4. **拼音+解释兜底**：确实没有通行英译的中国本土概念，用拼音加括号解释（如 qiyun (spirit resonance)）
5. **标注不确定性**：如果找到的译法不确定是否通行，在论文中标注"本文将X译为Y"，让读者知道这是作者的翻译选择

**对于查询预处理脚本**：脚本内置了本表的关键词映射，用于将中文搜索词自动转为中英双语查询。如果脚本查不到映射，会保留原文搜索，不会丢失任何查询。
